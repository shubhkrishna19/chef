# Quarterly Investors Report

A professional financial report dashboard with two viewing modes.

---

## Files

| File | Purpose |
|------|---------|
| `presentation.html` | **3D Experience** - Immersive Three.js presentation for investors |
| `index.html` | Standard dashboard with editor |
| `Investors_Report.md` | Markdown backup |

---

## Quick Start

1. Open `presentation.html` for the **3D immersive experience**
2. Open `index.html` for the **standard dashboard with editor**
3. Scroll to navigate through sections
4. Click **Edit Data** to modify values
5. Click **Export CSV** to download editable spreadsheet

---

## Features

- **3D Particle Background** - Animated space with floating geometries
- **Scroll Animations** - Smooth reveal effects
- **Mouse Parallax** - Interactive camera movement
- **CSV Export** - Editable in Excel/Sheets
- **Auto-save** - Data persists in browser

---

## Export Format

The CSV export is fully editable and contains:
- Financial metrics (Revenue, EBITDA, Profit, etc.)
- Channel performance data
- Achievements and strategy text
